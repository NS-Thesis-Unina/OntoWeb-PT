sequenceDiagram
    actor U as Utente
    participant P as Pagina Findings (HTTP/Analyzer/Techstack)
    participant DG as DataGrid Findings
    participant SHTTP as HTTP Findings Service (listHttpFindings)
    participant SAN as Analyzer Findings Service (listAnalyzerFindings)
    participant STECH as Techstack Findings Service (listTechstackFindings)

    U->>DG: Cambia pagina o page size (nella barra di paginazione)
    DG->>P: onPageChange(page, pageSize)

    P->>P: Calcola offset = page * pageSize limit = pageSize mantiene eventuali filtri/params interni
    P->>P: loading = true

    alt Sottosezione = HTTP Findings
        P->>SHTTP: listHttpFindings({ offset, limit })
        SHTTP-->>P: { ids[], total, offset, limit }
    else Sottosezione = Analyzer Findings
        P->>SAN: listAnalyzerFindings({ offset, limit })
        SAN-->>P: { ids[], total, offset, limit }
    else Sottosezione = Techstack Findings
        P->>STECH: listTechstackFindings({ offset, limit })
        STECH-->>P: { ids[], total, offset, limit }
    end

    P->>P: loading = false
    P->>DG: setRows(map(ids -> { id })) setRowCount(total) aggiorna page/pageSize da offset/limit
    DG-->>U: Mostra nuova pagina di findings (spinner integrato usato solo durante il fetch)
